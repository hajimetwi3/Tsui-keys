# Changelog

All notable changes to Tsui keys will be documented in this file.

## [0.8.0] - 2026-04-28

### Initial public release

- ブラウザで完結するローカルパスワードマネージャー
- AES-256-GCM + PBKDF2-SHA256 (800,000 iterations) による vault 暗号化
- 単一 HTML 配布版(GitHub Releases)+ Cloudflare Pages 配信版の 2 系統
- vault マージ機能、CSV インポート(Tsui / Chrome / 汎用形式)、パスフレーズ生成器
- 日本語 / English 対応
- PWA 対応(Cloudflare Pages 配信時)
